...and the network is expanding all the time.

Other applications are currently being considered to set up digipeaters by providing TNCs, radios and technical advice. The resulting network is expanding rapidly particularly in the Eastern States. The ROSE system for linking and directing is now operating well. There is still much to be done to develop the UHF links in the system at higher speeds - 4800 and 9600 bps are currently being implemented.

The funds for these operations have been provided from the surplus accumulated from sales of packet equipment to members. All this is done by unpaid volunteers and your help would be welcomed.
